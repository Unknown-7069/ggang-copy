<b>#설치방법</b><br>
①실리 최상단 메뉴 중 우측세번째에 있는 주사위 세 개 아이콘누르고 [확장 프로그램 설치]<br>
②첫번째 입력칸에 <b>https://github.com/Unknown-7069/ggang-copy</b> 넣고 인스톨폴미(빨간버튼)<br>
ㄴ다시 확장프로그램 눌렀을 때 [깡갤 복사기]보이면 설치 완료<br>
<br>
<B>#기본 사용방법</b><br>
①메세지 복사 : 시작위치, 종료위치에 메세지 넘버(숫자)쓰고 '단순 복사'누르면 텍스트 박스에 자동 복사됨<br>
②태그 제거 : 상태창,추론 태그 일괄 자동 삭제<br>
③줄바꿈 정리 : 과한 줄바꿈 깔끔하게 정리<br>
④위 내용 복사 : 현재 텍스트 박스에 적힌 내용 복사(클립보드로)<br>
⑤txt 저장 : 파일로 저장하기<br>
<br>
<b>#편의 기능</b> **사용시 ON*<br>
①작성중인 메세지 태그제거 : 채팅 입력창에서 상태창,추론 태그 제거<br>
②마지막 메세지 삭제 : 깡통,유저할거없이 제일 마지막 메세지 1개 삭제<br>
③삭제후 재생성 : 마지막 메세지 삭제 후 응답 재생성<br>
ㄴ옵션 중 입력필드 선택시 채팅 입력창에 아이콘 생김<br>
<br>
<b>대필 기능</b> **사용시 ON*<br>
①대필 프롬프트(명령하기) : 여기에 적힌 내용을 대필시 AI가 참고함<br>
②임시 대필칸 사용시 : 기본 채팅 입력창밑에 임시 대필칸이 새로 생기고, 임시 대필칸에 적힌 내용은 ①과 함께 대필시 AI가 참고함<br>
③임시 대필칸 미사용시 : 기본 채팅 입력창에 적힌 내용은 ①과 함께 대필시 AI가 참고함<br>
