<b>#설치방법</b><br>
①실리 최상단 메뉴 중 우측세번째에 있는 주사위 세 개 아이콘누르고 [확장 프로그램 설치]<br>
②첫번째 입력칸에 <b>https://github.com/Unknown-7069/ggang-copy</b> 넣고 인스톨폴미(빨간버튼)<br>
ㄴ다시 확장프로그램 눌렀을 때 [깡갤 복사기]보이면 설치 완료<br>
<br>
<b>#기본 사용방법</b><br>
①메세지 복사 : 시작위치, 종료위치에 메세지 넘버(숫자)쓰고 '단순 복사'누르면 텍스트 박스에 자동 복사됨<br>
②태그 제거 : 상태창,추론 태그 일괄 자동 삭제<br>
③정리 : 과한 줄바꿈 깔끔하게 정리<br>
④위 내용 복사 : 현재 텍스트 박스에 적힌 내용 복사(클립보드로)<br>
⑤비우기 : 텍스트 박스 내용 전체 삭제<br>
⑥txt 저장 : 파일로 저장하기<br>
<br>
<b>대필 기능</b> <br>
①대필 프롬프트(명령하기) : 여기에 적힌 내용을 대필시 AI가 참고함<br>
②임시 대필칸 사용시 : 기본 채팅 입력창밑에 임시 대필칸이 새로 생기고, 임시 대필칸에 적힌 내용은 ①과 함께 대필시 AI가 참고함<br>
③임시 대필칸 미사용시 : 기본 채팅 입력창에 적힌 내용은 ①과 함께 대필시 AI가 참고함<br>
<br>
<b>대필 프리셋</b><br>
①프리셋 저장(자동) : 대필 기본 지시문, 제외 지시문(부정 프롬프트), 대필 전용 프로필을 하나의 프리셋으로 저장.<br>
②프리셋 전환 : 프리셋 선택시 해당 설정으로 즉시 전환. 마지막 선택한 프리셋이 활성 상태로 지속 유지됨<br>
③프리셋 편집 : 연필 아이콘 클릭 후 이름 변경 또는 삭제 가능 (기본 프리셋 제외)<br>
④프리셋 복사 : 프리셋 선택 후 '현재 프리셋 복사' 선택시 동일한 내용의 새 프리셋 생성<br>
⑤순서 변경 : 프리셋 선택 후 '프리셋 순서 변경' 선택시 순서 조정 가능<br>
⑥사용할 프로필 : 실리 태번 페이지 새로고침/'사용할 프로필' 우측에 있는 새로고침 아이콘을 누르면 현재 유저의 '연결 프로필'을 연동해옴. 대필 요청시 설정된 '연결 프로필'기준으로 응답 요청함.(API,모델,메인 프롬프트 등)<br>
ㄴ대필 프로필 별도 설정시, 기본 3회까지 자동으로 재시도하며(서버 에러 방지) 중단 버튼 누를경우 즉시 원래의 프로필로 원복됨<br>
<br>
<b>#편의 기능</b> <br>
①작성중인 메세지 태그제거 : 채팅 입력창에서 상태창,추론 태그 제거<br>
②마지막 메세지 삭제 : 깡통,유저할거없이 제일 마지막 메세지 1개 삭제<br>
③삭제후 재생성 : 마지막 메세지 삭제 후 응답 재생성<br>
ㄴ옵션 중 입력필드 선택시 채팅 입력창에 아이콘 생김<br>
<br>
<b>#메세지 정리</b> <br>
①메시지 숨기기/보이기 : 설정 범위 내의 메세지를 숨겨줌(눈 감김)<br>
②다중 메세지삭제 : 설정 범위 내의 메세지를 영구 삭제 (백업 필수!)<br>
<br>
<b>#기타</b> <br>
①고화질 프로필 사진 : 채팅 화면에 뜨는 프사를 고화질로 나오게 함<br>
②입력창에 조절점 없애기 : 채팅입력칸 우측하단에 뜨는 입력박스 사이즈 조절 핸들 없애줌<br>
③입력창 안내문 없애기 : 채팅입력칸 내부에 있는 안내문구 없애줌<br>
④디버그 모드 : 추적 로그 활성 (개발 전용)<br>
