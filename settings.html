📋 깡갤 복사기
메세지 범위 입력 후 단순 복사 버튼을 클릭하면 클립보드에 자동 복사&아래 텍스트박스에 해당 내용이 삽입됩니다. 미입력시 자동으로 첫번째 메세지/마지막 메세지로 설정됩니다.
대필 기능 사용
대필 기본 지시문 ⏳
대필 제외 지시문 ⏳
사용할 프로필 ⏳
* 실험적인 기능입니다.
위 3가지(기본 지시문 + 제외 지시문 + 프로필)이 모두 현재 선택된 프리셋에 함께 저장됩니다. 프리셋은 항상 마지막에 선택된 프리셋으로 유지됩니다.
프리셋 순서 변경
항목을 드래그하여 순서를 변경하세요.
대필 버튼 위치
대필 아이콘()을 누르면, 위에 써진 내용(프롬프트)와 채팅창에 적힌 대필 지시문의 내용을 조합하여 사용자를 대신해 봇이 글을 써줍니다. (비어있는 곳은 알아서 무시합니다)
임시 대필칸 사용
체크하면 대필 전용칸이 생기고 기본 입력창 내용은 무시됩니다.
체크 해제시 기본 입력창에 쓴 내용을 대필 지시문으로 사용합니다.
메시지 숨기기/보이기
/hide와 /unhide 명령어를 사용해 메시지를 숨기거나 다시 보이게 합니다.
숨긴 메시지는 AI가 인식하지 못하지만 채팅 기록에는 남아있습니다.
버튼 클릭 시점 기준으로 확인합니다. (실시간 갱신 아님)
다중 메세지삭제
지정된 범위의 메시지를 영구적으로 삭제합니다.
실행 전 반드시 백업하세요.

기능을 켠 후 표시할 위치를 선택하세요.
복사기: 상단 소메뉴(기타 버튼 옆)에 아이콘 추가
입력필드: 채팅 입력창에 아이콘 추가
퀵메뉴 사용
퀵메뉴를 활성화하면 채팅 화면에서 빠르게 복사기 기능에 접근할 수 있습니다.
접근 방식
마법봉: 화면 우측 하단의 Extensions 메뉴(🪄)에서 접근
입력필드 아이콘: 채팅 입력창 옆에 아이콘 추가
표시할 섹션
체크 해제한 섹션은 퀵메뉴에서 숨겨집니다.
고화질 프로필 사진
프로필 사진을 고화질로 표시합니다.
이 옵션을 이용하는 것보다 SillyTavern의 config.yaml 파일에서 thumbnails 섹션의 enabled 값을 true에서 false로 변경하는 게 여러모로 훨씬 좋습니다. 파일 수정이 어려운 이용자를 위한 옵션입니다. 구형 기기를 사용할 경우 페이지 로딩 후 몇 초간 기니피그 한마리만큼의 성능 저하가 발생하며, 그 이후로는 햄스터 꼬리털 세포만큼의 성능 저하가 있습니다.
입력창에 조절점 없애기
채팅 입력창의 크기 조절 핸들을 제거합니다.
입력창 안내문 없애기
채팅 입력창의 '메시지를 입력하거나 /? 도와주기 위해' 문구를 숨깁니다.
대필 임시 지시칸을 사용중일 때는 해당 칸의 안내문구도 함께 숨겨집니다.
디버그 모드
문제 해결을 위해 콘솔에 자세한 로그를 출력합니다. 이게 뭔지 모른다면 키지마세요! 근데 켜도 크게 상관은 없습니다. 개미 더듬이 세포 하나만큼의 성능 저하가 발생합니다.
